## üîß Quick Fixes

### If Something Breaks:
1. Check browser console (F12)
2. Validate HTML: https://validator.w3.org/
3. Test locally first
4. Check file paths are correct
5. Clear browser cache (Cmd+Shift+R on Mac)

### Common Issues:
**CSS not loading:**
- Check file name is exactly `styles.css`
- Make sure it's in the same folder as `index.html`

**JavaScript not working:**
- Check file name is exactly `script.js`
- Look for errors in console (F12)

**Links not working:**
- Make sure URLs start with `http://` or `https://`
- Test each link after deployment
